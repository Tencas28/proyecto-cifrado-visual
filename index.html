                 Ayuda - Modos de Operación
            

            

                

                    
 Modo Codificación

                    
Convierte texto plano a código hexadecimal y luego a colores.

                    
    
                        
  • Entrada: Texto normal (ej: "Hola Mundo")
    • 
                        
  • Proceso: Texto → Hexadecimal → Colores
    • 
                        
  • Salida: Código hexadecimal y paleta de colores
    • 
                    

                

                
                

                    
 Modo Decodificación

                    
Convierte código hexadecimal a texto original.

                    
    
                        
  • Entrada: Código hexadecimal (ej: "486f6c61204d756e646f")
    • 
                        
  • Proceso: Hexadecimal → Texto → Colores
    • 
                        
  • Salida: Texto original y paleta de colores
    • 
                    

                

                
                

                    
 Consejos:

                    
    
                        
  • El código hexadecimal debe ser válido (solo caracteres 0-9, A-F)
    • 
                        
  • No incluyas el prefijo "#" en el código hexadecimal
    • 
                        
  • La longitud del hexadecimal debe ser múltiplo de 2
    • 
                        
  • Puedes pegar hexadecimal desde otros sistemas
    •
